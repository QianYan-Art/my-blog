Ubuntu Server Docker 部署 Firefox 完整指南

2025-11-204 min
c-仅记录

本指南用于在 Ubuntu 服务器（或其他 Linux 发行版）中通过 Docker 部署一个带有 Web 界面（NoVNC）的可操作 Firefox 浏览器。（jlesage/firefox镜像）

适用场景：远程办公、网络调试、挂机脚本、通过服务器访问内网资源。

1. 前置准备

确保服务器已安装 Docker。如果你的用户没有 Docker 组权限，请在所有命令前加上 sudo。

创建数据持久化目录

为了防止重启容器后书签、历史记录和登录状态丢失，我们需要在宿主机创建一个配置文件夹。

sudo mkdir -p /opt/docker/firefox/config

2. 启动容器 (最佳实践命令)

此命令已包含以下优化：

  • 高性能：分配 4GB 共享内存，防止浏览器崩溃。
  • 中文支持：启动时自动安装 font-noto-cjk（思源黑体）。
  • 高清画质：默认分辨率设为 1920x1080。
  • 安全加固：设置访问密码。

请修改下方命令中的 你的访问密码 后执行：

sudo docker run -d \
    --name=firefox \
    -p 5800:5800 \
    -v /opt/docker/firefox/config:/config:rw \
    --shm-size 4g \
    -e DISPLAY_WIDTH=1920 \
    -e DISPLAY_HEIGHT=1080 \
    -e INSTALL_DEPENDENCIES=font-noto-cjk \
    -e VNC_PASSWORD=你的访问密码 \
    --restart always \
    jlesage/firefox

参数详解

  • -p 5800:5800: Web 访问端口。
  • --shm-size 4g: 关键参数，解决现代浏览器内存不足崩溃的问题。
  • -e INSTALL_DEPENDENCIES=font-noto-cjk: 自动安装中文字体包，解决方块乱码。
  • -v ...: 将容器内的配置映射到服务器硬盘，保证数据不丢。
⚠️ 注意：运行命令后，请等待 1-2 分钟，容器需要时间在后台下载并安装字体包。

3. 防火墙设置

如果无法访问，请检查防火墙：

  1. 云厂商安全组（阿里云/腾讯云/AWS）：在控制台放行 TCP 5800 端口。
  1. 面板防火墙（1Panel/宝塔）：在防火墙设置中放行 5800 端口。
  1. UFW (Ubuntu原生): sudo ufw allow 5800/tcp

4. 浏览器初始化设置

访问地址：http://你的服务器IP:5800

A. 优化字体显示 (必做)

为了获得最佳的中文阅读体验：

  1. 打开 Firefox Settings (设置) -> 搜索 Fonts。
  1. 点击 Advanced (高级)。
  1. Proportional (比例字体): 修改为 Sans-serif (无衬线体)。
  1. Sans-serif: 选择 Noto Sans CJK SC。
  1. Monospace: 选择 Noto Sans Mono CJK SC。

B. 开启无痕/隐私模式 (可选)

如果你不希望浏览器记录历史网址：

  1. Settings -> Privacy & Security -> History。
  1. 选择 Never remember history。
  1. 按提示重启浏览器（Restart Firefox）。

C. 侧边栏工具使用

点击浏览器左侧的小箭头打开 NoVNC 控制栏：

  • Clipboard: 用于本机与服务器之间的文字复制粘贴中转。
  • Quality: 建议拉满（最右侧），获得最清晰画质。
  • Scaling Mode: 建议选 Local Scaling，让画面适应你的浏览器窗口。

5. 维护与管理

停止与启动

# 停止
sudo docker stop firefox

# 启动
sudo docker start firefox

# 重启 (如果感觉卡顿)
sudo docker restart firefox

彻底删除 (卸载)

如果你不想用了，想要彻底清除环境：

步骤 1：删除容器

sudo docker rm -f firefox

步骤 2：删除镜像 (释放 400MB+ 空间)

sudo docker rmi jlesage/firefox

步骤 3：删除配置文件 (慎用！会丢失所有书签和数据)

只有在你确定连数据都不要了的时候才执行：

sudo rm -rf /opt/docker/firefox

6. 常见问题排查

  • Q: 打开全是方块字乱码？

- A: 可能是自动下载字体失败。执行手动修复命令：

        sudo docker exec firefox apk update
        sudo docker exec firefox apk add font-noto-cjk
        sudo docker restart firefox
