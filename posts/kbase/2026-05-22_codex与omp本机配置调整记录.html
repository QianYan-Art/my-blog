11. Codex Documents Skill 本地显示修复

本机发现一处 Codex App 的 skill 引用显示问题：

  • Presentations 与 Spreadsheets 能在对话里正常标蓝
  • documents 同类引用不标蓝

本地排查结果：

  • documents 插件自身 plugin.json 的展示名是 Documents
  • 但对应 SKILL.md 头部原先写的是：

    • - name: documents

  • 同目录下另外两个 runtime skill 的显示名与 SKILL.md 头部名大小写一致：

    • - Presentations - Spreadsheets

因此本机做了一处最小修复：

  • 文件：

    • - C:\Users\lenovo\.codex\plugins\cache\openai-primary-runtime\documents\26.521.10419\skills\documents\SKILL.md

  • 改动：

    • - name: documents → name: Documents

用途：

  • 尽量让 Documents 的 skill 名与插件展示名保持一致
  • 提高 Codex App 将该 skill 正确识别为可点击蓝色引用的概率

注意：

  • 这是对 openai-primary-runtime 插件缓存做的本地补丁
  • 后续 runtime 升级后，可能被覆盖
  • 若升级后问题复发，可再次检查这处大小写是否被回退
