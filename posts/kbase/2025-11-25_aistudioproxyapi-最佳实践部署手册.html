服务器记录

1. 安装

2025-11-256 min

用户: qianyan (非 root)

核心原则: 严禁混用 pip 和 poetry；生产环境使用 poetry install 锁定版本。

1. 环境准备

如果是一台全新的服务器，需要先安装基础工具。

1.1 安装 Poetry (官方脚本)

不要使用 apt install python3-poetry，版本太老。

# 1. 安装
curl -sSL [https://install.python-poetry.org](https://install.python-poetry.org) | python3 -

# 2. 添加环境变量 (写入 .bashrc 永久生效)
echo 'export PATH="/home/qianyan/.local/bin:$PATH"' >> ~/.bashrc
source ~/.bashrc

# 3. 验证
poetry --version

1.2 安装 PM2 (如果未安装)

npm install pm2 -g

2. 项目部署 (全新安装)

2.1 克隆与配置

cd ~
git clone [https://github.com/CJackHwang/AIstudioProxyAPI.git](https://github.com/CJackHwang/AIstudioProxyAPI.git)
cd AIstudioProxyAPI

# 关键配置：让虚拟环境生成在项目目录下 (.venv)，避免权限问题
poetry config virtualenvs.create true
poetry config virtualenvs.in-project true

2.2 安装依赖

使用 install 而不是 update，确保与作者环境完全一致。

poetry install

2.3 下载浏览器内核

必须使用 Poetry 环境内的 Python 来下载，确保版本匹配。

poetry run python -m camoufox fetch

2.4 还原备份 (如果有)

如果你之前备份了登录信息：

# 假设备份在 ~/auth_profiles_backup
cp -r ~/auth_profiles_backup/* ~/AIstudioProxyAPI/auth_profiles/

2.5 启动服务 (PM2)

使用虚拟环境解释器启动，并开启无头模式。

pm2 start launch_camoufox.py \
  --name aistudio \
  --interpreter ./venv/bin/python \
  -- \
  --headless

3. 稳定性加固 (守护与监控)

3.1 设置定时重启

防止内存泄漏，设置每小时重启一次。

pm2 restart aistudio --cron "0 * * * *"

3.2 部署看门狗 (Watchdog)

防止 503 或 页面丢失 错误导致服务假死。

创建脚本 ~/watchdog.py:

(此脚本位于主目录，防止项目重装时丢失)

import subprocess
import time
import os

# 配置部分
LOG_FILE = "/home/qianyan/.pm2/logs/aistudio-error.log"
TARGET_PM2_NAME = "aistudio"
KEYWORDS = [
    "503", 
    "AI Studio 页面丢失", 
    "Locator expected to be enabled",
    "Unexpected server error"
]
COOLDOWN = 60

def restart_pm2():
    print(f"检测到关键错误，正在重启 {TARGET_PM2_NAME} ...")
    os.system(f"pm2 restart {TARGET_PM2_NAME}")
    time.sleep(COOLDOWN)

def follow(file):
    f = subprocess.Popen(['tail', '-F', '-n', '0', file], stdout=subprocess.PIPE, stderr=subprocess.PIPE)
    while True:
        line = f.stdout.readline()
        if line:
            yield line.decode('utf-8', errors='ignore')

if __name__ == "__main__":
    print(f"正在监视日志: {LOG_FILE}")
    try:
        if not os.path.exists(LOG_FILE):
            print(f"错误：日志文件不存在 {LOG_FILE}")
            exit(1)
        for line in follow(LOG_FILE):
            for keyword in KEYWORDS:
                if keyword in line:
                    print(f"捕获错误: {line.strip()}")
                    restart_pm2()
                    break
    except KeyboardInterrupt:
        print("停止监视。")

启动看门狗:

pm2 start ~/watchdog.py --name log-watcher --interpreter python3
pm2 save

4. 更新维护 (一键脚本)

⚠️ 警告: 不要手动执行 poetry update。请使用以下脚本同步更新代码、依赖和内核。

4.1 创建更新脚本 ~/update_aistudio.sh

#!/bin/bash
PROJECT_DIR="$HOME/AIstudioProxyAPI"

echo "=== 开始更新流程 ==="
cd "$PROJECT_DIR" || exit

echo "1. 暂停服务..."
pm2 stop aistudio

echo "2. 拉取最新代码..."
git pull

echo "3. 同步依赖 (Strict Mode)..."
# --sync 会删除本地多余的包，强制与 lock 文件一致
poetry install --sync

echo "4. 更新/匹配浏览器内核..."
poetry run python -m camoufox fetch

echo "5. 重启服务..."
pm2 restart aistudio

echo "=== 更新完成，正在检查日志 ==="
sleep 3
pm2 logs aistudio --lines 10 --nostream

4.2 赋予权限与修复格式

# 赋予执行权限
chmod +x ~/update_aistudio.sh

# 修复 Windows 换行符问题 (如果你是在 Windows 编辑后上传的)
sed -i 's/\r$//' ~/update_aistudio.sh

4.3 执行更新

以后需要更新时，直接运行：

~/update_aistudio.sh

5. 卸载与清理

如果你想彻底删除项目（例如重装）：

5.1 备份数据

cp -r ~/AIstudioProxyAPI/auth_profiles ~/auth_profiles_backup

5.2 删除进程

pm2 stop aistudio log-watcher
pm2 delete aistudio log-watcher

5.3 删除文件与缓存

注意: 必须删除 camoufox 缓存，否则下次安装可能会调用旧内核导致版本冲突。

rm -rf ~/AIstudioProxyAPI      # 删除项目
rm -rf ~/.cache/camoufox       # 删除浏览器内核缓存
# rm -rf ~/.cache/pypoetry     # (可选) 删除 pip 包缓存，通常建议保留以加速下次安装
